Design Diary

I. Where Shall I Begin?
The first entry in the Baptism at Bull Run design diary seeks to answer the most fundamental of questions: where does one start a project of this nature? This entry reveals the genesis of Baptism at Bull Run.
II. Essayons ("Let us try")
The second entry in the Baptism at Bull Run design diary uncovers the original source map used to generate the game's map and provides a glimpse at some of its details.
III. Cuboids!
The third entry in the Baptism at Bull Run design diary reviews the various units available in the game. In addition to infantry, cavalry, and artillery units, a unit unique to this game is revealed.
IV. In the Field: Securing Approaches & In Reserve
The fourth entry in the Baptism at Bull Run design diary reviews some of the details of the map specific to positioning units on the game board and introduces the field commands used in the game.
V. "There is Jackson standing like a stone wall!"
The fifth entry in the Baptism at Bull Run design diary introduces the three staff commands that are available in the game, some which affect an army's morale when so commanded.
VI. Putting It Together
The sixth entry in the Baptism at Bull Run design diary presents the remaining two phases of the game and outlines all four phases of a complete game turn.
VII. Esprit de Corps
The seventh entry in the Baptism at Bull Run design diary reviews the single, trackable game metric (as well as the mechanisms which affect it) by which one player's army defeats the other.
VIII. An Eye Fatiguing Number of Hours
The eighth (and final) entry in the Baptism at Bull Run design diary unveils the hours involved in taking this project from a germ of an idea to reality.
